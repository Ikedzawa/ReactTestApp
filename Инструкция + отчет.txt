Для старта прода приложения:

npm install -g serve
serve -s build

Отчет:

В приложении были имплементированы базовые функции "ToDo" листа.
- Добавление нового дела;
- Завершение дела;
- Удаление дела;

Подмечены особенности React, процессов разработки и архитектуры приложений.
Проведены параллели с Angular, исследованы интересные моменты.
